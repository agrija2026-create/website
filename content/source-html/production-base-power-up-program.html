産地生産基盤パワーアップ事業とは？補助率2分の1・上限と対象（機械・施設）を解説

2026年6月26日更新

産地生産基盤パワーアップ事業は、収益力強化に計画的に取り組む産地へ、高性能な機械・施設の導入や栽培体系の転換などをまとめて支援する農林水産省の補助事業です。「自分の産地は対象になるのか」「何に、どのくらい補助が出るのか」が気になる方に向けて、この記事では対象になる取組と担い手、補助の早見、補助率（定額・2分の1以内）と上限額の考え方、名前の似た強い農業づくり交付金との違い、協議会・都道府県・地方農政局を通じた申請の流れまでを、令和7年度補正（令和8年度に実施）の内容で整理します。

産地生産基盤パワーアップ事業の概要

まず全体像を表で押さえます。対象・補助の出し方・申請の入口を、産地・JA・農業法人・自治体の目線で整理しました。

項目 内容
対象になる人 産地で収益力強化に取り組む農業者や、農業者が組織する団体（地域農業再生協議会など）です。新市場の獲得をめざす拠点事業者と、それに連携する産地も対象になります。都道府県・市町村・民間団体等が支援の経路に入る区分もあります。
対象になる取組 機械・集出荷施設の導入、農業用ハウス・果樹園・茶園の再整備、省力樹形や優良品種への転換、貯蔵・加工・物流拠点の整備、堆肥・緑肥を使った土づくりなどです。三つの対策（新市場獲得・収益性向上・生産基盤強化）に整理されています。
補助額・補助率 取組の区分により、補助率は定額または2分の1以内です。上限額は事業メニューごとに実施要綱で定められます。令和7年度補正予算額は80億円（8,000百万円）です。
申請の入口 新市場獲得対策の一部は国の公募で募集されます。収益性向上対策・生産基盤強化対策は、都道府県を通じた要望調査で受け付けられます。まず産地の協議会・市町村・都道府県へ相談するのが入口です。

対象になるのは誰の・どんな取組か

この事業は、ひとりの農業者が単発で機械を買うための補助ではありません。産地として収益力強化に計画的に取り組むことが土台にあり、その計画を実現するための機械・施設・栽培転換などをまとめて支援します。

対象になる主な担い手は次のとおりです。

  • 産地で収益力強化に取り組む農業者と、農業者が組織する団体です。地域でまとまって取り組むため、地域農業再生協議会などの協議会が事業の主体になる区分があります。
  • 新市場の獲得をめざす拠点事業者と、それに連携する産地です。貯蔵・加工・物流の拠点を整える事業者と、その拠点へ安定して出荷する産地が組んで取り組みます。
  • 支援の経路として、都道府県・市町村・民間団体等が事業実施主体に入る区分もあります。誰が受け皿になるかは取組の区分で異なります。

「個人でも申請できますか」という点は、産地・地域の取組として位置づけられるかどうかが鍵です。多くの区分は協議会や団体を通じて取り組むため、個人で取り組みたい場合も、まず地域の協議会や市町村に計画への位置づけを相談するところから始めます。

支援メニューの早見

支援内容は大きく三つの対策に分かれます。自分の取組がどこに当てはまるかを、次の早見でつかんでください。

対策主な取組主な対象者
新市場獲得対策貯蔵・加工・物流拠点の整備、連携産地の生産・出荷体制づくり、優良品種・省力樹形の導入や栽培転換の実証拠点事業者と連携産地、産地の農業者・団体
収益性向上対策農業機械の導入、集出荷施設の整備、ヒートポンプ等による省エネ化収益力強化に取り組む産地の農業者・団体
生産基盤強化対策ハウス・果樹園・茶園の再整備・改修、次世代への継承、堆肥・緑肥を使った土づくり産地の農業者・団体、地域

どの対策でも、産地の収益力を高めるという目的でつながっています。複数の取組を組み合わせる産地も少なくありません。

何に出るか

ここが最も気になる点です。補助率と上限額の考え方を、対策ごとに整理します。

補助率は定額・2分の1以内

補助率は取組の区分により、定額または2分の1以内です。機械・施設の整備のように費用が大きい取組は2分の1以内、実証や調整のような取組は定額、というように設計されています。実際の率は事業メニューごとに実施要綱で定められます。

補助率2分の1以内の場合、補助額は事業費の半分が目安です。残りは自己負担で、規模感は次のとおりです。

総事業費補助額（2分の1）自己負担
1,000万円500万円500万円
5,000万円2,500万円2,500万円
1億円5,000万円5,000万円

定額の取組は「事業費の半分」という考え方ではありません。上限額・対象経費はメニューごとに実施要綱で定められるため、該当メニューの要綱で確認してください。

上限額はメニューごとに要綱で決まる

上限額は事業メニューごとに実施要綱で定められます。同じ事業の中でも、拠点施設の整備と機械の導入では上限の考え方が変わるため、自分の取組がどのメニューに当たるかを確かめたうえで、該当する実施要綱とQ&Aで上限額・対象経費を押さえるのが確実です。令和7年度補正予算額は事業全体で80億円（8,000百万円）です。

三つの対策で何に出るか

新市場獲得対策では、新市場が求めるロット・品質に応えるための貯蔵・加工・物流の拠点施設や、拠点と連携する産地の生産・出荷体制づくりに出ます。あわせて、需要の変化に対応した優良品目・品種の導入、省力樹形の導入、栽培方法の転換、技術導入の実証など、産地を先導する取組も支援します。

収益性向上対策では、収益力強化に取り組む産地の計画を実現するための農業機械の導入や集出荷施設の整備に出ます。施設園芸の産地では、燃油への依存を減らし省エネを進めるためのヒートポンプ等の導入も対象です。

生産基盤強化対策では、農業用ハウスや果樹園・茶園の再整備・改修、次世代への円滑な継承に向けたマッチングに出ます。さらに、堆肥や緑肥を使った土づくりを実証的に進め、全国へ広げる取組も支援します。

なお、共同利用施設の再編・合理化はこの事業ではなく、別枠の「新基本計画実装・農業構造転換支援事業」で支援します。隣り合う制度なので、共同利用施設を整えたいときは支援の入口を取り違えないよう注意してください。共同利用施設の補助については共同利用施設の整備に使える補助の解説で整理しています。

申請・採択の流れ

申請の経路は対策によって異なります。自分の取組がどの対策かを見きわめ、対応する入口から進めるのが近道です。

  • 新市場獲得対策の一部は、国の公募ページで募集されます。公募の要領にしたがって計画を作り、応募します。
  • 収益性向上対策・生産基盤強化対策は、都道府県を通じた要望調査で受け付けられます。市町村・都道府県の窓口へ要望を出すところから始まります。

産地として取り組む場合は、地域の地域農業再生協議会や市町村で取組を計画にまとめ、都道府県・地方農政局と調整しながら進めます。公募・要望調査のどちらでも、パンフレットで全体像をつかんだうえで、該当メニューの実施要綱とQ&Aを読み合わせると抜け漏れが減ります。要望調査の日程や様式は年度ごとに変わるため、最新の案内をご覧ください。

産地に求められる成果目標

この事業は、補助を受けて終わりではなく、産地として成果を出すことが前提です。事業全体では次の目標を定めています。

  • 輸出の拡大：青果物・花き・茶の輸出額を伸ばすことに寄与します。農林水産物・食品の輸出額5兆円（2030年まで）という政府目標とつながっています。
  • 販売額の伸長：品質向上や高付加価値化などにより、事業を実施した年度の翌々年度までに販売額を10%以上増やすことをめざします。
  • 生産資源の維持・継承：ハウスや園地など産地の生産資源を次の世代へ維持・継承し、産地そのものを持続させます。

個別の取組ごとに設定する成果目標や測定方法は実施要綱で定められます。計画づくりの段階で、達成できる目標になっているかを確かめておくと安心です。

制度の背景と資金の流れ

分散した産地が単独で投資を続けるだけでは、輸出や加工・業務用など広がる需要に応えきれません。そこで、収益力強化に取り組む産地を後押しし、新市場の獲得と生産基盤の維持・継承を同時に進めるために設けられているのがこの事業です。

資金は、取組の区分ごとに経路が分かれます。農業者や農業者が組織する団体が直接の受け皿になる支援に加え、国から都道府県を経由する形態、民間団体等（都道府県・市町村を含む）を介する形態があります。収益性向上対策・生産基盤強化対策の一部は、基金管理団体を通じて実施される区分があります。複数年度にわたる取組を支えるための仕組みです。

産地づくりの補助金は、この事業のほかにもあります。生産基盤の整備を幅広く支援する強い農業づくり総合支援交付金の解説や、加工・業務用での国産シェアを取り戻す取組をまとめた加工・業務用の国産シェア奪還に向けた取組の解説もあわせてご覧ください。

よくある質問

産地パワーアップ事業の補助率はどのくらいですか

取組の区分により、補助率は定額または2分の1以内です。機械・施設の整備のように費用の大きい取組は2分の1以内、実証や調整のような取組は定額が中心です。実際の率と上限額は、事業メニューごとに実施要綱で定められます。

対象になるのは誰ですか

産地で収益力強化に取り組む農業者と、農業者が組織する団体です。新市場の獲得をめざす拠点事業者と、それに連携する産地も対象になります。取組の区分によっては、都道府県・市町村・民間団体等が支援の経路に入ります。

個人でも申請できますか

多くの区分は、産地の協議会や団体を通じた地域の取組として位置づけられることが前提です。個人で取り組みたい場合も、まず地域の協議会や市町村に相談し、取組を地域の計画に位置づけてもらうところから始めます。

どこに申請しますか

新市場獲得対策の一部は国の公募で募集されます。収益性向上対策・生産基盤強化対策は、都道府県を通じた要望調査で受け付けられます。まずは産地の協議会・市町村・都道府県の窓口へ相談してください。

令和7年度補正の予算規模はどのくらいですか

令和7年度補正予算額は80億円（8,000百万円）です。事業全体の規模であり、各取組への配分や上限額はメニューごとに定められます。

強い農業づくり総合支援交付金との違いは何ですか

名前が似ていて混同しやすいですが、別の制度です。産地生産基盤パワーアップ事業は、産地として収益力強化に計画的に取り組むことを土台に、機械・施設の導入や栽培転換などを定額・2分の1以内で支援します。一方、強い農業づくり総合支援交付金は、食料システム構築・産地基幹施設等・卸売市場等の3タイプで、産地基幹施設や共同物流拠点などの整備を支援する制度です。産地の収益力強化計画にひもづく機械・施設なら産地パワーアップ、産地の基幹施設や流通拠点の整備なら強い農業づくり交付金、と入口を見分けると整理しやすくなります。

次の一歩

自分の産地が対象になりそうだと感じたら、計画づくりが最初の一歩です。地域の地域農業再生協議会や市町村に、収益力強化の取組として相談してください。そのうえで、収益性向上・生産基盤強化の取組は都道府県の要望調査の窓口へ、新市場獲得の取組は国の公募の案内へ進みます。手続の入口が分からないときは、地方農政局に相談すると、どのメニューに該当するかを確かめながら進められます。

キーワード解説

三つの対策（三大対策）

産地生産基盤パワーアップ事業の支援メニューを整理した枠組みです。新市場獲得対策・収益性向上対策・生産基盤強化対策の三つに分かれ、産地の取組がどこに当たるかで補助の中身が変わります。

地域農業再生協議会

地域の農業者・市町村・農業団体などで構成し、地域の農業の取組をまとめる協議会です。産地としてこの事業に取り組むとき、計画づくりや事業の主体になることがあります。

基金管理団体

複数年度にわたる支援を安定して行うために、資金を管理して事業を実施する団体です。収益性向上対策・生産基盤強化対策の一部は、この基金管理団体を通じて実施される区分があります。
